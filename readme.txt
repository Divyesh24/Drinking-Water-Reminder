=> This is a simple but very useful application of Drinking water Reminder at every hour 
   or we can change time according to our needs.

=> Many times, we are working on Laptop/Desktop for a long time and we also know that we 
   should take a short break during work to achieve great consistency towards our work so every hour we should drink water so that we don't get panic or feel stress.

=> also, this application is useful to schedule reminders at every 45 min or one hour, 
   maximum one hour to drink the water of glass.

=> The National Academies of Sciences, Engineering, and Medicine determined that adequate 
   daily fluid intake is: About 15.5 cups (3.7 liters) of fluids for men. About 11.5 cups (2.7 liters) of fluids a day for women.

=> To run the application:
Note: - Before running, please set your time according to your needs
		e.g. for one hour,
		set 60*60 = 3600 sec in sleep 
-> In terminal :
	TO start exe file : pythonw .\reminder.py

	TO stop exe file : taskkill /IM pythonw.exe /F
